# Campus Placement Portal (MVP)

## Vision

Build a simple placement portal where students can register, view jobs, and apply, while administrators can post and manage jobs.

---

## Problem Statement

Many colleges lack a centralized platform where students can easily access placement opportunities.

---

## Target Users

- Students
- Placement Officers
- Recruiters

---

## Success Metrics

- Student Registration
- Number of Job Posts
- Number of Applications
- Successful Placements

---

# Product Backlog

| Epic | User Story | Priority |
|------|------------|----------|
| Authentication | As a student, I want to register so that I can access the portal. | High |
| Authentication | As a student, I want to login securely. | High |
| Job Management | As a student, I want to view available jobs. | High |
| Job Application | As a student, I want to apply for jobs. | High |
| Admin Dashboard | As an admin, I want to add jobs. | High |
| Admin Dashboard | As an admin, I want to edit jobs. | Medium |
| Admin Dashboard | As an admin, I want to delete jobs. | Medium |

---

# Acceptance Criteria

## Student Registration

- Registration form validates input
- User account created
- Success message displayed

## Login

- Valid credentials allow login
- Invalid credentials show error

## Job Listing

- Available jobs displayed
- Job details visible

## Apply

- Student can submit application
- Confirmation message shown

## Admin

- Admin can add jobs
- Admin can edit jobs
- Admin can delete jobs
